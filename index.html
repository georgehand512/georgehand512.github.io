Unreal Projects

Crypt Raider

Crypt Raider is a simple game where players navigate a crypt to steal treasure by solving puzzles, like placing a statue on a podium to reveal a secret passage. Players avoid traps by swapping objects to escape. The project focuses on key development tasks: Level design with modular assets and Lumen lighting. Mover component for door movement. Grab component for object interaction via Blueprints. Pressure plate component for object-triggers. Utilising essential C++ concepts like pointers, actor components, loops, and calling C++ functions from Blueprints.

  • Access Code (GitHub)

Toon Tanks

A tank game where players control a blue tank to move, aim, and shoot at enemy turrets that also have basic A.I. The project covers tank movement, firing projectiles, enemy turrets, health, damage, and destruction mechanics. It also has special effects like explosions, smoke, and sounds, followed by win/lose conditions with HUD displays. Focus was on implimenting these features using C++ classes and Unreal Engine's input mappings.

  • Access Code (GitHub)

Simple Shooter

A simple shooter game with a fully animated third-person character. The project covers player movement, aiming, shooting mechanics with particle effects, health, and AI death. Implementing win/lose conditions, restart levels upon death, and display messages. Key elements include animation, shooting architecture, enemy AI with behavior trees, and health management for both the player and AI.
